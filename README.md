Operators
===============================================
To pass some of the tests, you have to write your return statements in one line of code.

`sumOfNaturalNumbers(10) === 55 // "true"`

`modulusAndSum(7, 5) === 9 // "true"`

`logicalAnd(10, 11, 3) === true // "true"`

`charAtHalfLength("Food") === 'o' // "true"`
